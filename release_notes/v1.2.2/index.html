    ⚡Highlights

    Added configs for reasoning LLMs

    As reasoning LLMs show remarkable performance, we added ReasoningLLMConfig for general support.

    Config class LLMs
    BasicLLMConfig Most non-reasoning LLMs
  1. Llama4
  2. Qwen3-30B-A3B-Instruct-2507
  3. ...
    		4.
    ReasoningLLMConfig Most reasoning LLMs
  4. Qwen3-30B-A3B-Thinking-2507
  5. gpt-oss-120b
  6. ...
    		7.
    Qwen3LLMConfig Qwen3 hybrid thinking
  7. Qwen3-30B-A3B
  8. Qwen3-32B
  9. ...
    		10.
    OpenAIReasoningLLMConfig OpenAI API reasoning models
  10. "o" series (o1, o3, o4)
  11. ...
    		12.  
    from llm_ie.engines import OpenAIInferenceEngine, ReasoningLLMConfig, Qwen3LLMConfig

# Reasoning LLM
llm = OpenAIInferenceEngine(base_url="http://localhost:8000/v1", model="Qwen/Qwen3-30B-A3B-Thinking-2507", api_key="EMPTY", 
                            config=ReasoningLLMConfig(thinking_token_start="<think>", thinking_token_end="</think>"))

# Qwen3 hybrid thinking
llm = OpenAIInferenceEngine(base_url="http://localhost:8000/v1", model="Qwen/Qwen3-30B-A3B", api_key="EMPTY", 
                            config=Qwen3LLMConfig(thinking_mode=False, temperature=0.0, max_tokens=2048))

    Messages log now stores reasoning tokens

    In FrameExtractor.extract_frames, AttributeExtractor.extract_attributes, and RelationExtractor.extract_relations, passing return_messages_log=True will return messages log with reasoning tokens.

    # extract frames
frames, messages_log = extractor.extract_frames(text_content=note_text, verbose=True, return_messages_log=True)

import json
print(json.dumps(messages_log, indent=4))

"""
> ...
> {
>    "role": "assistant",
>    "content": "...",
>    "reasoning": "<think>... </think>"
> }
"""

    📣Features and Changes

    InferenceEngine.chat and InferenceEngine.chat_async methods interface update

    InferenceEngine.chat now returns Union[Dict[str, str], Generator[Dict[str, str], None, None]]. A dict {"reasoning": , "response": } or Generator {"type": , "data": }

    InferenceEngine.chat_async now returns Dict[str,str]. A dict {"reasoning": , "response": }
